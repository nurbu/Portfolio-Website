# nurbu-portfolio

Personal portfolio site built with Next.js 15 and TypeScript. Deployed on Vercel.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Stack

- [Next.js 15](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vercel](https://vercel.com/) for deployment

## Deploy

Push to GitHub and import the repo in Vercel. It'll detect Next.js automatically and deploy with zero config.
